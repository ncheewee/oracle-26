HOW TO READ THISMODEL EDGE ≠ CERTAINTY

EV / value asks whether the odds are generous versus model probability. It can be high even for unlikely long shots.

Signal strength asks whether model confidence is strong enough to care. Green means price + conviction align; amber means speculative value; red means no-bet/avoid.

Default sort uses balanced value: signal strength first, then expected return. This avoids blindly ranking fragile long-shot EV above more dependable opportunities.
